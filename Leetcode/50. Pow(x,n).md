## [Problem](https://leetcode.com/problems/powx-n/description/)
Implement [pow(x, n)](http://www.cplusplus.com/reference/valarray/pow/), which calculates `x` raised to the power `n` (i.e., `xn`).

**Example 1:**

**Input:** x = 2.00000, n = 10
**Output:** 1024.00000

**Example 2:**

**Input:** x = 2.10000, n = 3
**Output:** 9.26100

**Example 3:**

**Input:** x = 2.00000, n = -2
**Output:** 0.25000
**Explanation:** 2-2 = 1/22 = 1/4 = 0.25

**Constraints:**

- `-100.0 < x < 100.0`
- `-231 <= n <= 231-1`
- `n` is an integer.
- Either `x` is not zero or `n > 0`.
- `-104 <= xn <= 104`


## My Approach
### Time: $O(n)$



```python
class Solution:
    def myPow(self, x: float, n: int) -> float:
        # x^11 = ((x^2)^2)^2*x^3
        # x^512 = (((x^2)^2)^2)^2...

        nn = int(n)
        ans = 1
        if nn < 0 :
            nn = -nn

        while nn:
            if (nn & 1 == 1):
                ans *= x
                nn -= 1
            else:
                x = x * x
                nn = nn >> 1

        if n < 0:
            return 1.0 / ans
        return ans
```
